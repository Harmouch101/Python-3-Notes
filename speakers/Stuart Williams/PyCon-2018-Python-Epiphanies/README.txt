Jupyter Notebook files for Python Epiphanies
presented at PyCon 2018 by Stuart Williams

Files Python-Epiphanies-[0-7]*.ipynb are separate Jupyter notebooks.

Python-Epiphanies-All.ipynb is the entire tutorial in one file.  It
will use a lot of memory and may be sluggish or cause your browser to crash.

This is an updated and reorganized version of the Python Epiphanies
O'Reilly video at http://shop.oreilly.com/product/0636920039815.do
